Receiver Functions

In Go, receiver functions allow methods to be defined on types. This feature is fundamental to Go's approach to object-oriented programming. Receiver functions can be defined for both value receivers and pointer receivers.

Example: Value Receiver

package main

import "fmt"

// Define a struct type
type Rectangle struct {
    width  int
    height int
}

// Define a method with a value receiver
func (r Rectangle) Area() int {
    return r.width * r.height
}

func main() {
    rect := Rectangle{width: 10, height: 5}
    fmt.Println("Area:", rect.Area())
}
Output: 
Area: 50

Example: Pointer Receiver

package main

import "fmt"

// Define a struct type
type Circle struct {
    radius float64
}

// Define a method with a pointer receiver
func (c *Circle) SetRadius(r float64) {
    c.radius = r
}

func (c Circle) Area() float64 {
    return 3.14 * c.radius * c.radius
}

func main() {
    c := Circle{}
    c.SetRadius(5)
    fmt.Println("Area:", c.Area())
}
Output: 
Area: 78.5

Common Data Structures

Here are some common data structures in Go along with examples:

Example: Slice

package main

import "fmt"

func main() {
    numbers := []int{1, 2, 3, 4, 5}
    fmt.Println("Numbers:", numbers)
}
Output: 
Numbers: [1 2 3 4 5]

Example: Map

package main

import "fmt"

func main() {
    capitals := map[string]string{
        "France": "Paris",
        "Japan":  "Tokyo",
    }
    fmt.Println("Capitals:", capitals)
}
Output: 
Capitals: map[France:Paris Japan:Tokyo]

Example: Struct

package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    p := Person{Name: "Alice", Age: 30}
    fmt.Println("Person:", p)
}
Output: 
Person: {Name:Alice Age:30}

Loops in Go

Introduction

Loops in Go provide mechanisms for iterating over collections or executing code repeatedly based on certain conditions. Go offers several types of loops to accommodate different programming needs.

Basic Loops

1. For Loop

The for loop in Go is versatile and supports various forms:

package loops

import "fmt"

func BasicLoops() {
    // Normal for loop
    for i := 0; i < 10; i++ {
        fmt.Println(i)
    }

    // While loop equivalent
    j := 0
    for j < 10 {
        fmt.Println(j)
        j++
    }

    // Infinite loop (break condition needed to exit)
    k := 0
    for {
        fmt.Println("Infinite loop")
        k++
        if k == 5 {
            break
        }
    }

    // Using range to iterate over collections (arrays, slices, maps, etc.)
    nums := []int{1, 2, 3, 4, 5}
    for index, value := range nums {
        fmt.Printf("Index: %d, Value: %d\n", index, value)
    }

    // Skipping iteration with continue
    for m := 0; m < 5; m++ {
        if m == 3 {
            continue
        }
        fmt.Println("Skipping 3:", m)
    }
}

Explanation:

  • Normal For Loop: Executes a block of code repeatedly until a condition is met (i < 10), with an increment or decrement (i++).
  • While Loop Equivalent: Uses a for loop with only a condition (j < 10), behaving like a while loop in other languages.
  • Infinite Loop: Demonstrates an infinite loop with for {}, usually used with a break condition (if k == 5) to exit.
  • Range Loop: Iterates over elements of a collection (nums slice) using the range keyword, providing both index and value for each iteration.
  • Skipping Iteration with continue: Skips iterations based on a condition (if m == 3) using continue.

Key Features of Go's Loop Constructs:

  • No do-while Loop: Go does not have a do-while loop, but you can simulate it with a for loop and a break condition.
  • Range-Based Iteration: Simplifies iterating over arrays, slices, maps, and channels with the range keyword.
  • Versatility: The for loop in Go is highly flexible, supporting traditional loop constructs and specialized forms like iterating over collections with range.

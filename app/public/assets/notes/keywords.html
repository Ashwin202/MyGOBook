Commonly Used Keywords in Go

Keyword Purpose
chan Declares a channel.
const Declares a constant.
defer Schedules a function call to be run after the function completes.
for Creates a loop.
func Defines a function.
go Starts a new goroutine.
if Creates a conditional statement.
import Imports packages needed for the code.
interface Declares an interface type.
new Allocates memory and returns a pointer.
package Defines the package to which the file belongs.
panic Stops the ordinary flow of control and begins panicking.
recover Regains control of a panicking goroutine.
return Returns from a function.
select Waits on multiple channel operations.
struct Declares a structure type.
type Defines a new type.
var Declares a variable.
